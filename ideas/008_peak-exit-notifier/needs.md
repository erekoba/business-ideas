# ニーズ調査

## 市場規模

- **市場規模の概算**: 日本国内のターゲット市場規模は約300〜500万人、潜在売上規模は年間約6〜15億円と推定。

  - 根拠: 国内家庭用ゲームユーザー数は約3,200万人（ファミ通2023年調査）。うち「タイパ重視の社会人ゲーマー」層を20代〜40代の就労者に絞ると約1,500万人。さらに「積みゲーを抱えている」「プレイ時間の最適化に関心がある」層を20%と見積もると約300万人。フリーミアム転換率3〜5%（有料ユーザー9〜15万人）×月額300〜500円×12ヶ月で年間約3〜9億円。広告・アフィリエイト収益を加算すると上限15億円圏内に収まる。

- **成長トレンド**: **拡大基調（追い風3つ）**

  1. **タイパ意識の社会的浸透**: 2022〜2024年にかけて「タイムパフォーマンス」という概念が若年層を中心に定着。映像の倍速視聴・要約サービスの普及がゲーム領域にも波及しつつある。
  2. **積みゲー問題の深刻化**: Steamのセール文化・PlayStation Plus・Nintendo Switch Onlineによるタイトル供給過多が常態化。「遊びきれない」ユーザー層が構造的に拡大している。
  3. **ゲーム実況視聴層の拡大**: 「プレイせずに視聴で済ませる」選択肢が一般化しており、「どこまでプレイすれば十分か」という問いへの需要が顕在化しやすい土壌が育っている。

- **根拠・参考データ**:
  - ファミ通ゲーム白書2023: 国内ゲームユーザー数・年齢別分布
  - 博報堂生活総合研究所「タイパに関する生活者調査」2023
  - Steam Spy推計・国内Steamユーザー数（推定800万人超）
  - HowLongToBeat月間訪問者数（SimilarWeb推計: 全世界月間約1,500万UU、日本からの流入は約3〜5%）

## 顧客ニーズの検証

- **顧客が今どう解決しているか（代替手段）**:

  | 代替手段 | 利用シーン | 限界 |
  |----------|-----------|------|
  | HowLongToBeat参照 | 「あと何時間かかるか」を確認し、終わりを見極める | クリアまでの残量はわかるが「もうやめていいか」の判断は自分でしなければならない |
  | ゲーム実況視聴に切り替え | 「疲れてきたからYouTubeで続きを見る」 | 自分のプレイ体験の感情ピークは回収できない。代替消費に過ぎない |
  | Steamレビュー・5chスレ参照 | 「ここからつまらなくなる？」をコミュニティに聞く | 主観の平均値であり、自分のピーク位置と一致しない。都度検索コストが高い |
  | 積みゲー化（事実上の放棄） | 「面白くなくなったら自然にやめる」 | 罪悪感が残る。「やめた」ではなく「積んだ」という自己評価の歪みが生じる |
  | Notionや手書きでプレイ日記 | 感情の変化を自己記録する | 継続コストが高く、ほとんどのユーザーが3日以内に離脱する |

- **不満・ペインポイント**:

  1. **「もったいない」呪縛**: お金を払ったゲームを途中でやめることへの罪悪感。特に定価5,000〜9,000円のタイトルで顕著。サンクコストバイアスの典型。
  2. **やめる「許可」がもらえない**: 「面白くないな」と感じても、「あと少しで面白くなるかも」という期待が離脱を妨げる。外部からの客観的シグナルがない。
  3. **ピークを言語化できない**: 「あのあたりが一番楽しかった」という感覚はあるが、それを記録・可視化する手段がなく、後から振り返れない。
  4. **積みゲーの罪悪感が次のゲームへの着手を妨げる**: 「前のゲームを終わらせてから」という心理が、新しい体験への移行コストを上げている。
  5. **コンテンツ過多による選択疲れ**: 次に何をやるかの判断コストが高く、惰性で同じゲームを続けるほうが楽になっている。

- **支払い意欲があるか**:

  **中程度（月額300〜500円なら転換可能な層が存在するが、無料前提の文化との摩擦あり）**

  - ゲームツール・トラッカー系アプリは無料が当然視される傾向が強く、単独での有料転換は難易度が高い。
  - ただし「時間を最適化するツール」というフレーミングであれば、生産性ツール（Notion, Obsidianなど）への課金習慣がある層（推定30〜40万人規模）に訴求しやすい。
  - 支払いトリガーとして有効なのは「感情曲線の長期履歴保存」「複数タイトルの比較分析」「Saturation Point到達通知のカスタマイズ」など、記録資産が蓄積されるほど価値が上がる機能設計。
  - ゲームギフト券・Amazonギフト等との連携ポイント還元型であれば無料ユーザーのエンゲージメント維持に有効。

## 競合分析

| 競合名 | 強み | 弱み |
|--------|------|------|
| HowLongToBeat | 世界最大のクリア時間UGCデータベース・高い認知度・無料で使える安心感 | 「いつやめるか」の判断支援ゼロ・感情トラッキング機能なし・クリア完遂前提の設計思想 |
| Backloggd / Steam記録機能 | プレイ履歴・評価の記録・SNS共有機能・既存プラットフォームとの連携 | 行動変容を促す設計なし・「積んだ」という事実の記録に終始・撤退の肯定的フレーミングなし |
| GameTrack（iOS） | UI/UXの洗練度・積みゲー可視化・モバイルファースト設計 | 感情曲線・飽和点の概念なし・コレクション管理ツールの域を出ない |
| YouTube/ニコニコゲーム実況 | ピーク部分だけの切り取り視聴が事実上可能・無料・コンテンツ量が膨大 | 自分のプレイ体験の感情追跡不可・受動的消費であり能動的撤退判断を支援しない |
| Notion/手書き日記 | 自由度の高さ・既存ツールで完結・カスタマイズ性 | 継続コストが高い・通知機能なし・フォーマット不統一でデータとして機能しない |

## 差別化ポイント

**1. カテゴリーの先占による概念独占**
「ゲームからの撤退を勝ち逃げと定義する」フレーミングは現時点で競合ゼロ。技術的優位ではなく「問いの定義権」を持つことが最大の参入障壁になる。「Saturation Point」という造語自体がブランド資産になりうる。

**2. 感情曲線の個人データ資産化**
ユーザーが記録を積み重ねるほど「自分だけのプレイ感情履歴」が生まれ、他サービスへの乗り換えコストが上昇する。客観的クリア時間データベース（HowLongToBeat）とは根本的に異なる個人化された価値を提供できる。

**3. UXの「冷徹さ」による記憶への刻み込み**
既存ゲームツールはすべて「楽しさの継続」を前提に設計されている。「Saturation Point到達」という無機質で断定的な通知は、ゲームツール史上前例のないUX体験であり、口コミ・スクリーンショット共有の自然な動機になる。

**4. 006（TTV可視化）との両輪設計**
入口（いつ面白くなるか）と出口（いつやめるか）を一つのサービスコンテキストで提供できるのは本アイデアだけ。単体では成立しにくい「ゲームライフ最適化」という上位概念を独占し、競合が模倣困難な文脈を構築できる。

**5. コンプリート至上主義への逆張りによる拡散力**
「クリアを目指すな」というメッセージは既存ゲームコミュニティの価値観に正面から対立する。批判・議論を生むことがそのまま認知獲得につながるカウンターカルチャー設計であり、広告費ゼロでの初期拡散が期待できる。

## 結論

- **ニーズの強さ**: **中**

- **理由**:
  ペインポイント（積みゲーの罪悪感・サンクコストの呪縛・惰性プレイによる時間の浪費）は実在し、ターゲット層の日常的な悩みとして確認できる。ただし「強」ではなく「中」と評価する理由は以下の3点。

  第一に、**ニーズが潜在的であり自覚されていない**。「ゲームをやめるタイミングを教えてほしい」と能動的に検索するユーザーはほぼ存在しない。課題は存在するが、それをプロダクトで解決するという発想がユーザー側にないため、需要を顕在化させるための教育・啓発コストがかかる。

  第二に、**支払い意欲の壁**。ゲームツールの無料前提文化は根強く、感情トラッキングという抽象的な価値に月額課金させるまでのUX設計難度が高い。無料フェーズでのエンゲージメント構築が収益化の前提条件になる。

  第三に、**行動変容の難しさ**。「面白くないと思いながらも続ける」という行動はサービスが存在しても変わらない可能性がある。通知を受け取っても「あと少しだけ」と続けるユーザーが多数派になるリスクがあり、プロダクトの本質的な価値提供がUI設計の巧拙に強く依存する。

  これらを踏まえると、**コンセプトの独自性と拡散ポテンシャルは高いが、収益化までの距離は中程度以上**と見積もるのが現実的な評価となる。006との連携による文脈の厚みを早期に構築することが、単体での限界を超えるための最優先戦略となる。
