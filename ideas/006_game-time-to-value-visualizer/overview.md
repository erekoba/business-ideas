# ゲームのTime to Value可視化サービス

## 一言説明

ネタバレなしで「そのゲームがいつ・どれだけ面白くなるか」を時系列グラフで可視化し、忙しい社会人ゲーマーの「始める勇気」を支援するWebサービス。

## ターゲット顧客

- **対象**: 20〜40代の忙しい社会人ゲーマー・積みゲー持ち・タイパ重視層（潜在共感層480〜720万人）
- **抱えている課題**: 「50時間かかるゲームに踏み出せない」「面白さを調べようとするとネタバレを踏む」という二律背反。クリア時間はわかっても「その時間が均質に楽しいのか、序盤が苦行なのか」が既存サービスでは一切わからない。
- **提供する価値**: 最初の「鳥肌ポイント」到達時間・システム全開放時間・中だるみ区間の有無を、ネタバレゼロで時系列グラフとして提示。「このゲームは3時間耐えれば化ける」という感情言語化フォーマットで、始める/やめる判断を合理的に下せるようにする。

## ビジネスの概要

既存のゲーム情報サービス（HowLongToBeat・Metacritic・Steamレビュー・GameWith）はすべて、「ネタバレあり」か「点数のみ」のどちらかに偏っており、「ネタバレゼロ×面白さの時系列構造の可視化」を両立するサービスは国内外に存在しない。本サービスはその構造的空白地帯を狙う。

収益モデルはハイブリッド型。無料閲覧で集客しつつ、詳細チャートを月額300円のサブスクで提供するフリーミアム構造と、Steam/Amazon経由のアフィリエイト成果報酬を組み合わせる。副業フェーズでは初期投資5,000〜15,000円（自作前提）、月間固定費2,000〜4,000円と極めて低コストで始められる。

収益シミュレーションは保守的に見て開始3〜6ヶ月で月3万円・12〜18ヶ月で月20万円・24ヶ月以降で月75万円が目安。名目黒字は3〜6ヶ月、労働コストを含む実質黒字は12〜18ヶ月が現実的な到達ラインとなる。

最大のリスクは「プレイ時間確保」と「ネタバレなしという品質基準維持」の二重制約によるコンテンツ拡充速度の上限。対策として、初期はプレイ済みタイトル30〜50本をストック型コンテンツとして積み上げ、段階的にユーザー投稿型へ移行することで労働依存度を下げる設計が有効。まずプレイ済み3本を記事化してSNSに無料投稿し、2週間以内に反応を計測してからサイト構築投資の判断をするアプローチが最もリスクを抑えられる。
