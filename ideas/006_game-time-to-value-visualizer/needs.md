# ニーズ調査

## 市場規模

- **市場規模の概算**: 日本のゲーム人口は約5,200万人（ファミ通2023年調査）。そのうちPCゲーム（Steam）ユーザーは推定600〜800万人。本サービスのコアターゲットである「忙しい社会人ゲーマー・積みゲー持ち・タイパ重視層」はPC/コンシューマー問わず広く存在し、20〜40代のゲーマー人口（約2,400万人）の少なくとも20〜30%（480〜720万人）が潜在的な共感層と推定される。月額300円課金の有料転換率を一般的なフリーミアムSaaSの2〜5%と仮定すると、有料ユーザーの上限は10〜35万人規模。現実的な初期狙い目は1,000〜5,000人。

- **成長トレンド**: 追い風が複数存在する。①「タイパ（タイムパフォーマンス）」という概念が2022〜2023年にかけて若年層・社会人層に急速に定着（新語・流行語ノミネート）。②Steamの日本語対応タイトル増加と国内PCゲーマー人口の緩やかな拡大。③積みゲー問題のSNS言及数が継続的に増加しており、X（旧Twitter）での関連ハッシュタグ投稿は年々増加傾向。④ゲームパスやPS Plus等のサブスク普及により「とりあえず積む」行動が加速しており、消化に向けた意思決定支援ニーズが構造的に拡大している。

- **根拠・参考データ**: ファミ通ゲーム白書2023（国内ゲーム人口）、Steam Spy推定データ（国内Steamアクティブユーザー）、電通「タイパに関する意識調査」2023、X検索による「積みゲー」関連投稿量（月間数万件規模で確認可能）、HowLongToBeatの月間訪問者数（SimilarWeb推定：全世界月間1,000〜1,500万PV）を参考に日本語市場の潜在需要を類推。

## 顧客ニーズの検証

- **顧客が今どう解決しているか（代替手段）**: 主に以下の4通り。①Steamレビュー欄を流し読みしてネタバレを踏みながら判断する。②YouTubeで序盤プレイ動画を視聴して雰囲気をつかむ（ただしネタバレリスクあり・時間消費が大きい）。③Twitterでフォロワーの感想をクロール（属人的・サンプル偏り大）。④「どうせ積む」と諦めて購入・インストールを放置する。いずれも根本的な解決策になっておらず、ネタバレを避けながら客観的な体験構造を把握する手段は現状存在しない。

- **不満・ペインポイント**: ①最大のペインは「ネタバレを踏まずに序盤の辛さが続くのかどうかを知る手段がない」こと。②クリア時間（HowLongToBeat）はわかっても「その50時間が均質に楽しいのか、序盤30時間が苦行なのか」が不明。③ゲームの面白さが「序盤に出にくい」タイトル（JRPG、ソウルライク、シミュレーション等）で特にペインが深く、このジャンルのファン層はターゲットと高度に一致する。④「やめる判断」もできずに放置→罪悪感という積みゲーループが心理的ストレスになっている。

- **支払い意欲があるか**: 条件付きで「中〜高」と推定。根拠は三点。第一に、月額300円はNetflixやゲームパスに慣れた層には心理的ハードルが低い価格帯であり、ゲーム1本の購入判断を誤るコスト（数千円〜数万円＋数十時間）と比較すれば合理的と感じやすい。第二に、HowLongToBeatのPatreonサポーターが数千人規模存在することは「無料で使えるゲーム情報サービスに自発的に課金するユーザー層」の存在証明になっている。第三に、タイパ重視層は「時間の節約」に対して金銭的価値換算をしやすく、「1本のゲーム選択ミスを防げるなら300円は安い」という論理が刺さりやすい。ただし、無料代替手段が多い現状では「なぜこれにお金を払うか」の価値訴求が弱いと離脱率が高くなるリスクがある。

## 競合分析

| 競合名 | 強み | 弱み |
|--------|------|------|
| HowLongToBeat | ユーザー数・プレイ時間データが圧倒的、Steam連携済み、グローバル認知 | クリア時間のみで面白さの時系列変化は非対応。ネタバレコメント混入リスクあり |
| Metacritic / OpenCritic | メディア・ユーザーレビュー集約、信頼性・ブランド力 | スコアのみで「いつ面白くなるか」不明。ネタバレコメント多数 |
| GameWith / 攻略まとめサイト | SEOドメイン力・日本語対応・広告収益基盤が確立済み | ネタバレ前提の設計。タイパ・心理的ハードル解消の視点は皆無 |
| Steamレビュー欄 | 手軽・量が多い・購入直前に参照できる | ネタバレ混入が避けられない。体験の時系列構造が読み取れない |
| Backloggd | 積みゲー管理・コレクション機能・コミュニティ感 | 「面白さ曲線」の可視化機能なし。英語圏中心で日本語対応弱い |

## 差別化ポイント

「ネタバレゼロ」と「面白さの時系列可視化」を同時に満たすサービスは国内外を通じて現時点で存在しない。既存競合はいずれかしか満たしておらず、本サービスが狙うポジションは構造的な空白地帯になっている。

具体的に勝てる理由は三点ある。第一に、**編集設計そのものが参入障壁**になる。「ネタバレなしで面白さの構造を言語化する」という作業は高度な編集スキルと設計思想を要するため、大手競合が「やろうと思えばできるが、やる理由がない」状態であり、追随コストが高い。第二に、**「3時間耐えれば化ける」という感情言語化フォーマット**はデータの羅列より拡散されやすく、SNSでのバイラル獲得に適している。ゲームコミュニティはXやRedditでの言及が購買行動に直結しやすい文化を持っており、拡散起点になるコンテンツフォーマットとの相性が良い。第三に、**忙しい社会人・積みゲー層という文脈への特化**により、攻略情報を求めるユーザーとの自然な棲み分けが生まれ、既存の攻略系メディアとの正面衝突を回避できる。競合が「やり込みたい人向け」に設計されているのに対し、本サービスは「始める勇気を持てない人向け」という入口を押さえる設計であり、ゲームライフサイクルの異なるフェーズを取る。

## 結論

- **ニーズの強さ**: 中

- **理由**: ペインポイント自体は実在しており、「ネタバレを踏まずに体験の構造を知りたい」という欲求はSNSやゲームコミュニティで継続的に観測できる。競合が空白地帯を放置しているという構造も確認されており、コンセプトとしての独自性は高い。ただし「高」ではなく「中」にとどめる理由は二点ある。第一に、現時点ではユーザーが「このサービスで調べる」という行動習慣が存在しておらず、潜在ニーズを顕在化させるための認知獲得コストが未知数である。代替手段の組み合わせ（YouTube＋Twitter＋Steamレビュー）で「なんとなく対処している」ユーザーが多く、積極的に新サービスを探しに来るモチベーションが形成されていない。第二に、課金転換の根拠となる「無料版で価値を体験する」フローがまだ設計されておらず、月額300円の支払い意欲が実際の行動につながるかはMVP検証なしには断言できない。市場の存在を自ら証明しながらPMFを探る必要があるフェーズであり、ニーズの確からしさはあるが確定はしていない、という評価が現時点では正確である。
