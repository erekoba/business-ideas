# 資料HTML化＆ワンクリック共有サービス

## 一言説明

テキストやMarkdownを貼るだけでAIが見栄えの良いHTMLに変換し、権限管理付きのURLで即座に共有できる、非エンジニア向けの資料公開サービス。

## ターゲット顧客

- **対象**: 社内勉強会の主催者・フリーランス・小規模チームのリーダー・コミュニティ運営者など、エンジニアではないが「ちゃんとした見た目で情報を共有したい」と思っている人（日本国内、推定500万人規模）
- **抱えている課題**: Google DocsやNotionでは見た目が資料らしくならない。HTMLを生成AIで作れても、デプロイ・URL発行・権限設定が難しくて結局諦める。PDFはスマホで崩れる。Gamma.appは日本語精度が低い。
- **提供する価値**: 「貼る→HTML化→URL発行→権限設定」を3ステップ・5分以内で完結させる。非エンジニアがWebサーバの知識ゼロでも、見栄えの良い資料を特定の人だけに届けられる。

## ビジネスの概要

生成AI（OpenAI API等）を活用し、ユーザーが入力したテキスト・Markdown・箇条書きを美しいHTMLに自動変換。変換後はそのままサービス上でホスティングし、閲覧用URLを即発行する。URLにはパスワード保護・招待制・閲覧期限・アクセスログといった権限管理機能を付与できるため、「不特定多数に公開したくないが特定の人には見せたい」という閉じた共有ニーズに対応する。

収益モデルはフリーミアム型サブスクリプション。無料プランで月3ページまで提供し、個人プラン（月額680円）・チームプラン（月額1,980円）への転換を狙う。主力ARPUは800円前後を想定。

固定費は月1万円前後（クラウドサーバ＋API費用）と極めて軽く、損益分岐点は有料ユーザー14人。月3万円の副業ラインに必要な有料44人は、SNS・技術ブログ・勉強会コミュニティへの継続発信によってリリース後3〜5ヶ月での達成が現実的な射程距離にある。

直接競合はGamma.appのみだが、日本語精度・限定共有特化・低価格の3点で明確な差別化が可能。NotionやGoogleがこの領域を本格強化した場合は即座に競合化するリスクがあるため、「日本語ネイティブ×閉じた共有×超低摩擦の操作体験」というポジションを早期に確立することが最重要課題となる。
