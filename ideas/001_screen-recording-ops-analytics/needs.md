# ニーズ調査

## 市場規模

- **市場規模の概算**: 
  - 国内BPR（業務プロセス改善）コンサルティング市場：約3,000億円規模（2023年）
  - 国内RPA・業務自動化関連市場：約1,200億円（2024年、ITR調べ）
  - 本サービスが狙う「業務可視化ツール」セグメント：上記市場の前工程として推定50〜200億円規模
  - 直接的なターゲット企業数：従業員100名以上の企業 約27,000社（総務省統計）、うち業務改善予算を持つ企業は推定30〜40%（約8,000〜10,000社）

- **成長トレンド**: 
  - DX推進・生産性向上は政府方針として継続的に強化
  - リモートワーク普及により「目に見えない業務」の可視化ニーズが拡大
  - AIによる画像・動画解析コストの低下が追い風
  - 2025年以降、中小企業へのDX補助金拡充によりターゲット層が拡大見込み

- **根拠・参考データ**: 
  - 経済産業省「DXレポート2.2」（2022）：業務プロセス改善を課題とする企業は回答企業の68%
  - ITR「国内業務プロセス管理市場」レポート
  - 矢野経済研究所「RPA市場調査2023」

## 顧客ニーズの検証

- **顧客が今どう解決しているか（代替手段）**: 
  - 現場へのヒアリング（最も一般的・最もコストがかかる）
  - 業務観察（張り付き・シャドーイング）
  - マニュアル・手順書の精読
  - Excelでの自己申告式工数管理
  - タイムトラッキングツール（Toggl、TimeCrowdなど）の導入
  - コンサルティング会社への委託

- **不満・ペインポイント**: 
  - ヒアリングは「建前」を話されやすく実態と乖離する
  - 観察は心理的プレッシャーを与え、通常業務と異なる行動を引き起こす
  - 自己申告工数は入力の手間と不正確さが問題
  - マニュアルは実際の作業と乖離していることが多い
  - コンサル委託は費用が高く（数百万〜数千万）、中小企業には敷居が高い
  - 「何をどこから改善すればいいかわからない」状態からのスタートが最も辛い

- **支払い意欲があるか**: 
  - 高い。業務改善コンサルに数百万払っている企業が現状の顧客層
  - 月20〜50万円/社であれば、10名以上の現場調査コストの代替として合理的
  - 稟議が通りやすい：「可視化ツール」はDX投資として予算化されやすいカテゴリ

## 競合分析

| 競合名 | 強み | 弱み |
|--------|------|------|
| Teramind（米国） | 従業員監視・生産性分析機能が充実、エンタープライズ実績豊富 | 英語UIメイン・日本語サポート弱い・監視色が強くプライバシー懸念 |
| Veriato（米国） | 詳細な操作ログ・コンプライアンス用途に強い | 日本市場展開が弱い・BPR用途より情報漏洩対策寄り |
| TimeCrowd / Toggl | 導入容易・コスト低い | 自己申告ベースで実態乖離・画面解析なし |
| 日立ソリューションズ「業務可視化ソリューション」 | 大手信頼性・国内サポート | 高価格（数百万〜）・中小企業には過剰 |
| UiPath Process Mining | プロセスマイニング機能が強力 | 導入ハードル高・基幹システムログが前提でPCスクリーンは弱い |
| Notion / Confluence（業務文書化ツール） | 普及率高い | あくまで文書化であり実態の自動キャプチャは不可 |

## 差別化ポイント

1. **日本語・日本業務に特化**: 日本特有のExcel業務・社内システム（勘定奉行・弥生等）への対応
2. **監視ではなく改善が目的のUI/UX設計**: 「監視ツール」ではなく「改善支援ツール」として従業員の心理的抵抗を下げる
3. **中小〜中堅企業向けの低価格帯**: 大手コンサルや海外エンタープライズツールが手薄な層を狙う
4. **初期はレポート納品型でSaaS化前のPMF検証**: 小回りが利くサービス設計

## 結論

- **ニーズの強さ**: 高
- **理由**: 業務改善の「最初の一歩」である現状把握は、あらゆる企業が課題としており、既存手段のどれも完全には解決できていない。DX推進予算の取りやすさとも相まって、支払い意欲の高い顧客が一定数存在する。
