# ニーズ調査

## 市場規模

- **市場規模の概算**: 日本国内の潜在市場規模は年間**約6〜15億円**と推定。内訳：国内ゲーム配信者（YouTubeゲーム系チャンネル＋Twitch国内配信者）は推定**約30〜50万人**。そのうち切り抜き需要が発生する月1万再生以上の中規模以上配信者を**約3〜5万人**と絞り込む。加えて、収益目的の専業・副業切り抜き師を**約1〜2万人**と推定。月額2,000円のサブスクを想定した場合、3万人が利用すれば年間**約7.2億円**、5万人なら**約12億円**規模。切り抜き師×配信者の双方が対象のため、市場アドレス可能範囲は単一ユーザー層より広い。

- **成長トレンド**: **拡大基調**。YouTube Shortsの収益化開始（2023年〜）により切り抜き縦型動画の投稿インセンティブが増大。VTuber市場の拡大（2023年国内市場規模約800億円超・矢野経済研究所推計）に伴いゲーム配信コンテンツ総量が増加中。切り抜きチャンネル数はYouTubeで可視化できるだけでも直近2年で倍増傾向にある。

- **根拠・参考データ**: YouTubeの国内月間利用者数約7,000万人（Google公表・2023年）／Twitchの国内配信者数推定10〜20万人／VTuber市場規模調査（矢野経済研究所・2023年）／YouTube Shorts収益化プログラム開始による縦型動画投稿量の急増（Google公式発表）／クラウドワークス・ランサーズにおける「切り抜き編集」案件の検索件数増加傾向（各プラットフォーム公開データより定性推定）

## 顧客ニーズの検証

- **顧客が今どう解決しているか（代替手段）**:
  - **切り抜き師**: 配信アーカイブを全編視聴または早送りしながら手動で名シーンを特定 → DaVinci Resolve・Premiere Proで編集 → テロップ・サムネを手作業で追加。1本あたり**2〜5時間**が相場。
  - **配信者本人**: TwitchのClips機能・YouTubeの切り抜き許可設定で視聴者任せにするか、自身でショート動画を手動編集して投稿。大半は「やりたいが時間がない」として放置。
  - **VTuber事務所**: 社内スタッフまたは外注切り抜き師に発注。コスト・納期・品質のばらつきが課題。

- **不満・ペインポイント**:
  1. **時間コストが最大の障壁**: 3時間の配信から5分の切り抜きを作るために2〜4時間かかるという非効率。副業切り抜き師にとって時給換算が低く収益性が悪い。
  2. **名シーン発見の主観的困難さ**: どこが「バズるシーン」か判断するために全編視聴が必要。スキップすると見落としが生じるジレンマがある。
  3. **テロップ・縦横比変換の二度手間**: 編集後にさらにShorts・TikTok向けのフォーマット変換作業が別途発生する。
  4. **配信者との連絡コスト**: 切り抜き許可・素材受け取り・フィードバック対応など、編集以外の調整コストが見えないところで積み上がる。

- **支払い意欲があるか**: **あり（条件付き）**。切り抜き師は月5〜10本を稼働すれば1本あたり500〜3,000円の収入があり、月額2,000〜3,000円のツールへの投資は作業時間削減で十分に回収できる計算が成立する。配信者本人は「自分で切り抜きを出したい」意欲はあるが時間がなく、月1,000〜2,000円程度の手軽な価格帯なら試用する層が一定数いる。ただし**「精度が低ければすぐ解約する」というスイッチングコストの低さがリスク**であり、無料トライアルによる体験価値の先行提供が購買転換の鍵となる。

## 競合分析

| 競合名 | 強み | 弱み |
|--------|------|------|
| Opus Clip | 自動切り抜き生成の先行実績・洗練されたUI・英語圏での大規模ユーザー基盤 | 日本語処理精度が低い・ゲーム配信文脈の理解がない・コメント欄解析機能なし |
| Clipping Factory | 日本語対応・国内VTuber事務所との取引実績・国内事業者としての信頼感 | 個人向け価格が不透明・AI自動化の深度が浅く手動工程が残る・スケーラビリティ不明 |
| Adobe Premiere Pro（自動リール機能） | ブランド信頼性・総合編集機能の充実・既存ユーザーへのバンドル効果 | ゲーム配信特有の名シーン検出ロジックなし・専門知識が必要・切り抜き特化ではない |
| 人力切り抜き師（クラウドソーシング） | 人間的な文脈理解による質の高い判断・配信者との関係構築 | 納期が遅い・スケール不可・費用が本数比例で積み上がる・品質のばらつき |
| TwitchClips / YouTube切り抜き機能 | 無料・視聴者が自発的に作成・導入ゼロ | 編集なし・テロップ・サムネなし・配信者がコントロールできない |

## 差別化ポイント

**①「日本語×ゲーム文脈×コメント欄」の三重特化による名シーン検出精度**
Opus Clipが手をつけていない「草」「神」「wwww」等の日本語スラングや、コメント欄の密度スパイクをゲーム配信の文脈で解釈する機能は、現時点で実質的な直接競合が不在。汎用ツールが「盛り上がり」を音声の大きさで検出するのに対し、本サービスはゲームUIイベント（キル表示・レベルアップ・ボス撃破演出）とコメントスパイクと配信者の感情を統合的に評価するため、ゲーム配信に限定すれば名シーン適合率で明確な優位を持てる。

**②切り抜き師の「作業完結」を目指すワンストップ設計**
名シーン検出だけでなく、テロップ自動生成・縦横比変換（16:9→9:16）・複数候補のスコアリング表示をひとつのフローで提供することで、「ツールを使った後にまだ別ソフトを開く」という現行の分断ワークフローを解消する。「数時間→15分以内」という時間削減の訴求は、副業切り抜き師の時給改善に直結するため費用対効果の説明がしやすい。

**③個人・副業ユーザーが手を出せる価格設計**
法人向け個別契約のClipping Factoryや月$19〜のOpus Clipに対し、月額1,000〜2,000円帯（またはShorts1本あたり数十〜百円の従量課金）という価格設定により、副業切り抜き師・個人配信者が「とりあえず試せる」心理的ハードルを下げる。

## 結論

- **ニーズの強さ**: **高**

- **理由**: 切り抜き作業の時間コストという明確で定量化されやすいペインが存在し、かつ切り抜き師・配信者双方に「解決したい」意欲と一定の支払い意欲が確認できる。市場はVTuber拡大・YouTube Shorts収益化によって今まさに拡大しており、ニーズが顕在化するタイミングと本サービスの参入タイミングが合致している。「日本語×ゲーム配信×コメント欄解析」の特化領域に直接競合が不在である点は、ニーズの強さに加えて**市場獲得の速度優位**にもなりうる。ただし、精度が期待値を下回った場合の解約率が高くなるリスクがあり、ニーズの強さを収益に変換するには**初期ユーザーとの密な検証サイクルによるMVP精度の早期引き上げ**が前提条件となる。
